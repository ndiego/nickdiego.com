## Notices (GitHub Alert Syntax)

Use GitHub-flavored alert syntax for callouts. Just write a blockquote starting with `[!NOTE]`, `[!TIP]`, `[!IMPORTANT]`, `[!WARNING]`, or `[!CAUTION]`:

> [!NOTE]
> This is a **note**. Use it for supplementary information readers should know.

> [!TIP]
> This is a **tip**. Use it for helpful advice on doing things better.

> [!IMPORTANT]
> This is **important** information that users need to achieve their goal.

> [!WARNING]
> This is a **warning**. Use it for urgent info requiring immediate attention.

> [!CAUTION]
> This is a **caution**. Use it to advise about risks or negative outcomes.

### JSX Syntax

You can also use the `<Notice>` component directly with the `type` prop (`note`, `tip`, `important`, `warning`, `caution`):

<Notice type="note">
This is a **note** using JSX syntax.
</Notice>

<Notice type="tip">
This is a **tip** using JSX syntax.
</Notice>

## Code Blocks

### Basic Code Block

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

### With Filename

```tsx filename="components/Button.tsx"
interface ButtonProps {
  children: React.ReactNode;
  onClick?: () => void;
  variant?: 'primary' | 'secondary';
}

export function Button({ children, onClick, variant = 'primary' }: ButtonProps) {
  return (
    <button
      onClick={onClick}
      className={variant === 'primary' ? 'bg-blue-500' : 'bg-gray-500'}
    >
      {children}
    </button>
  );
}
```

## YouTube Embed

Embed YouTube videos by their ID:

<YouTube id="dQw4w9WgXcQ" title="Example YouTube Video" />

With a start time (in seconds):

<YouTube id="dQw4w9WgXcQ" title="Video starting at 30 seconds" start={30} />

## Tweet Embed

Embed tweets by their ID:

<Tweet id="1354143047324299264" />

## GitHub Stats

<GitHubStats repo="WordPress/gutenberg" />

## Combining Components

Components can be combined within the same section. For example, you might show code alongside a notice:

<Notice type="note">
The following example requires React 19 or later.
</Notice>

```tsx filename="app/page.tsx"
export default function Page() {
  return (
    <main>
      <h1>Hello, Next.js!</h1>
    </main>
  );
}
```
